# Vue 3 + TypeScript + Vite

## 環境構築

1. Node のインストール

- https://nodejs.org/en/download
- 長期サポートがされている LTS をダウンロード

---

2. Yarn のインストール

```cl
npm install -g yarn
```

---

3. プロジェクトの作成

```cl
yarn create vite
```

- プロジェクト名を入力
- 言語とフレームワークの選択

---

4.  ライブラリのインストール

```cl
yarn
```

5. アプリケーションの起動

```cl
yarn dev
```
